# Legacy Historical Docs

`backtest/docs/legacy_historical/` 存放舊歷史資料方案的補充文件。這裡不是正式 runtime 或正式資料根目錄，而是歷史設計、舊流程與過渡說明的保存區。

## 資料夾內容

- `README.md`
  本文件。
- `DATA_DOWNLOADS_README.md`
  舊資料下載流程說明。
- `MODULE_ANALYSIS.md`
  舊模組分析文件。
- `QUICKSTART.md`
  舊快速開始指南。

## 使用邊界

- 這裡的文件只應拿來理解歷史背景、舊路徑與遷移脈絡。
- 若內容和目前 `backtest/data/`、`backtest/runtime/`、`backtest/README.md` 衝突，以目前正式主線為準。
- 不要把這個資料夾當成新的文件鏈起點；它屬於補充與封存資料。

## 文件層級

這個資料夾底下沒有更深一層的 README。其他 Markdown 文件由本文件統一收口。
