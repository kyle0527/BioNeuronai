### 🧠 生物啟發特性

#### Hebbian 學習機制
```
"同時激活的神經元會連接在一起"
- 無監督學習: Δw = η × x × y
- 有監督學習: Δw = η × (target - y) × x × (target + 0.1)
- 權重衰減: w = w - λ × w
- 穩定性約束: w ∈ [0, 2.0]
```

#### 新穎性檢測算法
```python
# 輸入新穎性 (餘弦相似度)
similarity = dot(current, previous) / (norm(current) * norm(previous))
input_novelty = 1.0 - similarity

# 輸出新穎性 (變異度)
output_novelty = min(1.0, variance(recent_outputs) * 4.0)

# 總新穎性
novelty = 0.7 * input_novelty + 0.3 * output_novelty
```
