---

### **步驟 1: 註冊幣安測試網帳號** 🎫
**✨ 只需執行一次**

#### 1.1 訪問測試網
```
網址: https://testnet.binancefuture.com
```

#### 1.2 登入選項
- **選項 A**: 使用 GitHub 帳號登入（推薦）
- **選項 B**: 使用 Google 帳號登入

#### 1.3 獲得虛擬資金
- ✅ 登入後自動獲得 **10,000 USDT** 虛擬資金
- ✅ 資金用完可以重置
- ✅ 完全模擬真實環境

#### 1.4 熟悉測試網介面
```
□ 查看帳戶餘額
□ 瀏覽交易對列表
□ 了解訂單類型
□ 查看持倉和歷史
```

---

### **步驟 2: 生成 API 金鑰** 🔑
**✨ 只需執行一次**

#### 2.1 進入 API 管理頁面
```
1. 點擊右上角個人頭像
2. 選擇 "API Keys" 或 "API 管理"
3. 點擊 "Create New Key" 或 "創建新密鑰"
```

#### 2.2 設置 API 權限
```
✅ 必須啟用: 讀取資訊
✅ 必須啟用: 現貨和槓桿交易
✅ 必須啟用: 期貨交易
❌ 不要啟用: 提現權限
```

#### 2.3 保存 API 憑證
```python
# 你會得到兩個密鑰：
API_KEY = "xxxxxxxxxxxxxxxxxxxxxx"  # 公開密鑰
API_SECRET = "yyyyyyyyyyyyyyyyyyyyyy"  # 私密密鑰

⚠️ 重要提醒：
- Secret 只會顯示一次，請立即保存
- 不要分享給任何人
- 測試網密鑰不能用於正式網
```

#### 2.4 測試 API 權限
```bash
# 使用 curl 測試連接
curl -X GET "https://testnet.binancefuture.com/fapi/v1/ping"

# 應該返回: {}
# 表示連接成功
```

---

### **步驟 3: 配置本地系統** ⚙️
**✨ 只需執行一次**

#### 3.1 編輯配置文件
```bash
# 打開配置文件
code config/trading_config.py
```

#### 3.2 填入 API 憑證
```python
# config/trading_config.py

# =======================================
# 幣安 API 配置
# =======================================

# 從步驟 2 複製的 API 憑證
BINANCE_API_KEY = "貼上你的 API Key"
BINANCE_API_SECRET = "貼上你的 API Secret"

# 確保使用測試網
USE_TESTNET = True  # ✅ 必須是 True

# =======================================
# 交易配置
# =======================================

# 監控的交易對
TRADING_PAIRS = [
    "BTCUSDT",   # 比特幣
    "ETHUSDT",   # 以太坊
]

# 自動交易（建議先關閉，手動測試）
AUTO_TRADE_ENABLED = False  # 先設為 False

# 使用策略融合
USE_STRATEGY_FUSION = True  # 使用 AI 三策略融合

# =======================================
# 風險管理
# =======================================

MAX_RISK_PER_TRADE = 0.02       # 單筆最大風險 2%
MAX_TRADES_PER_DAY = 10         # 每日最多 10 筆交易
MIN_SIGNAL_CONFIDENCE = 0.65    # 最低信號置信度 65%
MAX_DRAWDOWN_PERCENTAGE = 0.10  # 最大回撤 10%
```

#### 3.3 驗證配置
```bash
# 檢查配置是否正確
python -c "from config.trading_config import *; print(f'測試網: {USE_TESTNET}'); print(f'API Key 長度: {len(BINANCE_API_KEY)}'); print('✅ 配置完成' if len(BINANCE_API_KEY) > 0 else '❌ 請填入 API Key')"
```

### **步驟 4: 測試連接** 🔌
**✨ 只需執行一次（驗證設定是否正確）**

#### 4.1 測試基本連接
```bash
# 測試 API 是否能連接到幣安
python -c "
from src.bioneuronai.crypto_futures_trader import BinanceFuturesConnector
from config.trading_config import *

connector = BinanceFuturesConnector(
    api_key=BINANCE_API_KEY,
    api_secret=BINANCE_API_SECRET,
    testnet=USE_TESTNET
)

print('🔗 測試連接...')
price = connector.get_ticker_price('BTCUSDT')

if price:
    print(f'✅ 連接成功！')
    print(f'BTC 當前價格: ${price.price:,.2f}')
else:
    print('❌ 連接失敗')
"
```

**預期輸出**:
```

✅ 連接成功！
BTC 當前價格: $93,010.00

# 第一步：確定風險金額

# 第四步：確認倉位價值

```

```
A: 
1. 確認使用的是測試網帳號
2. 登入測試網頁面確認餘額
3. 如果需要可以重置測試帳戶

```
A:

4. 嘗試切換到波動更大的交易對
