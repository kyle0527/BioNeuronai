# Code of Conduct

## Our Pledge

We are committed to a respectful, harassment-free project environment for contributors and users.

## Expected Behavior

- Be direct and constructive.
- Focus feedback on code, evidence, and project goals.
- Respect different experience levels and backgrounds.
- Disclose conflicts of interest when discussing trading performance or financial claims.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information or credentials.
- Misrepresenting performance, security posture, or deployment readiness.
- Pressuring others to run live trading systems without understanding risk.

## Enforcement

Maintainers may remove comments, close issues, or block contributors who violate this code of conduct.
