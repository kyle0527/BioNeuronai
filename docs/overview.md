# Overview

BioNeuronAI is a bio-inspired neural computation toolkit that focuses on novelty gating, Hebbian learning, and responsible security integration for AI assistants. This documentation set complements the bilingual README by providing deeper architectural context and auto-generated API references.

## Highlights

- **Novelty-driven neurons** with configurable memory and learning parameters.
- **Composable networks and pipelines** for experimentation and production use.
- **Security-conscious modules** that harden AI assistants against injection and misuse.
- **Smart Learning Assistant** utilities that observe repository health and suggest improvements.

Refer to the `architecture.md` whitepaper for a deep dive into the design principles and to the API Reference for module-level details.
