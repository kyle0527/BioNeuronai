## 1. 核心願景

### 1.1 目標

建立一個系統，能夠：

1. **自我優化**: 每個策略用歷史數據調整自己的參數
2. **相互競爭**: 不同策略在相同條件下比較績效，優勝劣汰
3. **AI 融合創新**: 最終創造出「混搭」策略，例如：
   - 進場邏輯來自策略 A
   - 退場邏輯來自策略 B  
   - 參數設定來自策略 C
   - 風控規則來自策略 D

### 1.2 與傳統方法的差異

| 傳統方法 | 本系統 |
|----------|--------|
| 人工設計固定策略 | 策略自動進化 |
| 策略彼此獨立 | 策略基因可以混搭 |
| 參數人工調整 | 參數自動優化 |
| 單一策略交易 | AI 融合多策略優點 |

## 3. 基因設計

### 3.1 基因類型

| 基因類型 | 說明 | 範例選項 |
|----------|------|----------|
| **Entry Gene (進場)** | 決定何時進場 | MA交叉, RSI超賣/超買, MACD, 布林帶, 價格突破, 成交量突破 |
| **Exit Gene (退場)** | 決定何時退場 | 追蹤止損, 固定止盈, 回歸均值, 時間退場, 反向信號 |
| **Risk Gene (風控)** | 止損規則 | 固定%止損, ATR止損, 波動率止損, 資金曲線止損 |
| **Sizing Gene (倉位)** | 倉位大小 | 固定比例, Kelly公式, 波動率調整, 風險平價 |
| **Param Gene (參數)** | 各基因的參數 | 週期, 閾值, 乘數, 回看天數... |
