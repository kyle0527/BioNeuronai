# 社群活動規劃

BioNeuronAI 將於 2024 年下半年舉辦系列黑客松與工作坊，歡迎開發者、研究人員與企業團隊參與。

## 活動一覽

| 活動 | 日期 | 形式 | 目標對象 | 重點內容 |
| ---- | ---- | ---- | -------- | -------- |
| BioNeuronAI 黑客松 | 2024/08/23-25 | 線上 + 實體混合 | 開發者、資料科學家 | 建構新穎性閘門與安全模組的創新應用 | 
| 安全策略工作坊 | 2024/09/12 | 線上研討會 | 安全治理、法務、AI 產品經理 | 建立安全模組審核流程與最佳實踐 |
| 產業導入工作坊 | 2024/10/03 | 實體閉門 | 企業技術長、解決方案顧問 | 深入介紹 PoC 流程與部署架構 |

## 報名指南

1. 前往 [活動報名表](https://forms.gle/bioneuronai-events) 填寫基本資料與參與動機。
2. 黑客松組隊資訊將於活動前兩週公布，可自由組隊或個人報名。
3. 工作坊名額有限，將依報名順序與需求審核結果發送確認信。
4. 所有活動皆提供錄影回放與教材下載，請於報名表勾選是否需取得補課資源。
