## 一、三層策略進化系統（養蠱式遺傳演算法）

**獨特性** ⭐⭐⭐ | **可移植性** ⭐⭐⭐ | **商業價值** ⭐⭐⭐

### 核心概念

BioNeuronAI 設計了一套以「養蠱」為靈感的策略自動進化機制，讓交易策略在競爭環境中自然演化，無需人工介入即可持續最佳化。

### 三層架構

```
第1層：策略競技場 (Strategy Arena)
  └─ 100 個候選策略並行回測
  └─ 依夏普比率、勝率、最大回撤計算適應度分數
  └─ 每代淘汰最差 20%，保留最優 30% 進行繁殖

第2層：階段路由器 (Phase Router)
  └─ 識別 9 個交易時段（開盤、上午、中午、收盤…）
  └─ 為每個時段分配最優策略
  └─ 市場狀態變化時自動切換

第3層：組合優化器 (Portfolio Optimizer)
  └─ 遺傳演算法優化全局多時段策略配置
  └─ 最大化跨時段的風險調整後報酬
```

### Know-How 要點

- **交叉與突變設計**：策略參數以 50/50 單點交叉 + 15% 隨機突變率產生子代，平衡探索與利用
- **多目標適應度函數**：綜合夏普比率、Sortino 比率、最大回撤、勝率等多指標，避免過度最佳化單一指標
- **多進程並行加速**：100 個策略同時回測，大幅縮短每代進化時間
- **「養蠱」隔離機制**：策略之間不共享資源，模擬真實競爭環境

### 可移植場景

- 股票、外匯、期貨量化策略最佳化
- 任何需要超參數自動調整的 ML 系統
- 遊戲 AI 策略訓練

## 二、AI + 技術指標雙流融合推論引擎

**獨特性** ⭐⭐⭐ | **可移植性** ⭐⭐ | **商業價值** ⭐⭐⭐

### 核心概念

BioNeuronAI 不單純依賴 AI 預測或傳統技術指標，而是設計了一套**雙流加權融合架構**，動態合併兩者優勢。

### 架構設計

```
輸入市場數據
      ↓
┌─────────────────┐    ┌─────────────────────────────┐
│   AI 流 (40%)   │    │   技術指標流 (60%)            │
│                 │    │                             │
│ 1024維特徵提取  │    │ RSI 背離 + 布林帶 + MACD     │
│ 111.2M MLP模型  │    │ + EMA + ADX + ATR + 量能    │
│ 輸出: 信心分數  │    │ 輸出: 策略信號              │
└────────┬────────┘    └───────────────┬─────────────┘
         └──────────────┬──────────────┘
                        ↓
                  信號融合引擎
                  (加權投票 + 方向一致性過濾)
                        ↓
                  最終交易決策
```

### Know-How 要點

- **非對稱權重設計**：AI 佔 40%、技術指標佔 60%，經過回測驗證的最優比例，避免過度依賴黑盒模型
- **置信度門檻過濾**：AI 預測置信度 < 0.6 時，降低 AI 權重或拒絕交易，防止低信度訊號
- **方向一致性強化**：兩流方向一致時才執行交易，大幅降低假訊號率
- **即時推論速度**：~22ms 完成整個推論流程，滿足高頻交易需求

## 四、10 種市場狀態自動識別與動態路由

**獨特性** ⭐⭐⭐ | **可移植性** ⭐⭐⭐ | **商業價值** ⭐⭐⭐

### 核心概念

市場不同狀態下，同一策略的表現差異極大。BioNeuronAI 建立了 **10 種市場狀態分類器**，並根據當前狀態自動路由到最適合的策略。

### 10 種市場狀態

```
STRONG_UPTREND     ↗↗  強勢上漲   → 趨勢追隨策略（高槓桿）
UPTREND            ↗   上漲趨勢   → 趨勢追隨策略
WEAK_UPTREND       →↗  弱勢上漲   → 趨勢追隨策略（低倉位）
RANGING            ↔   橫盤震盪   → 均值回歸策略
WEAK_DOWNTREND     →↘  弱勢下跌   → 均值回歸 / 空單輕倉
DOWNTREND          ↘   下跌趨勢   → 空單趨勢策略
STRONG_DOWNTREND   ↘↘  強勢下跌   → 空單策略（高槓桿）
HIGH_VOLATILITY    ⚡   高波動     → 縮小倉位 / 觀望
BREAKOUT           💥   突破       → 突破交易策略
CONSOLIDATION      📦   整理       → 等待突破確認
```

### Know-How 要點

- **狀態轉換平滑機制**：防止市場狀態頻繁震盪導致過度交易，使用滯後確認
- **狀態 × 策略矩陣**：建立二維映射表，每種狀態有主策略和備選策略
- **動態倉位係數**：根據狀態強度（強勢/弱勢）自動縮放倉位大小
- **歷史狀態分布統計**：回測時分析各狀態出現頻率，驗證路由邏輯合理性

## 五、企業級四層風險管理框架

**獨特性** ⭐⭐ | **可移植性** ⭐⭐⭐ | **商業價值** ⭐⭐⭐

### 核心概念

BioNeuronAI 將風險管理系統化為多層防線，從倉位計算到即時監控，形成完整的風控鏈。

### 四大核心組件

#### 1. Kelly Criterion 動態倉位計算

```python
# 根據策略歷史勝率與賠率動態計算最優倉位
kelly_fraction = win_rate - (1 - win_rate) / profit_factor
position_size = capital * kelly_fraction * safety_factor
```

Know-How：使用半 Kelly（0.5× 係數）降低方差，並設置上限防止過度集中

#### 2. 六點交易前檢查清單

```
① AI 置信度 ≥ 0.6
② 今日回撤 < 最大回撤閾值
③ 今日交易次數 ≤ 上限
④ 可用餘額 ≥ 最小保證金
⑤ 無緊急風險警報
⑥ 槓桿倍數在允許範圍內
```

Know-How：全部通過才執行交易，任一失敗即拒絕，「先守後攻」原則

#### 3. 四級風險等級體系

| 等級 | 每筆風險 | 最大回撤 | 夏普下限 | 適用場景 |
|------|----------|----------|----------|----------|
| CONSERVATIVE | 1% | 5% | 0.8 | 保守投資者 |
| MODERATE | 2% | 10% | 0.6 | 一般投資者 ⭐ |
| AGGRESSIVE | 3% | 15% | 0.5 | 進取投資者 |
| HIGH_RISK | 5% | 20% | 0.3 | 高風險承受者 |

#### 4. 12 項統計指標即時追蹤

勝率、獲利因子、夏普比率、Sortino 比率、最大回撤、平均盈虧比、
連勝/連敗次數、期望值、卡瑪比率、月度收益分布、回撤持續時間、VaR

### Know-How 要點

- **風控與策略解耦**：風控模組獨立，可插拔到任何交易策略上
- **動態閾值調整**：根據市場波動率自動調整回撤閾值（高波動期放寬）
- **峰值追蹤算法**：高水位線追蹤，準確計算水下深度與持續時間

## 六、10 步交易 SOP 自動化控制器

**獨特性** ⭐⭐⭐ | **可移植性** ⭐⭐ | **商業價值** ⭐⭐

### 核心概念

將資深交易員的操作流程編碼為 **10 步標準作業程序 (SOP)**，由系統自動依序執行，消除人為情緒干擾。

### 10 步驟流程

```
步驟 1: 系統健康檢查（API 連線、資金、風控狀態）
步驟 2: 宏觀市場掃描（恐貪指數、全球市值、DeFi TVL）
步驟 3: 市場狀態識別（10 種狀態分類）
步驟 4: 交易對篩選（流動性、波動率、資金費率）
步驟 5: 信號生成（多策略並行分析）
步驟 6: 信號確認（方向一致性、強度驗證）
步驟 7: 倉位計算（Kelly Criterion + 風控限制）
步驟 8: 交易前檢查（6 點清單驗證）
步驟 9: 執行交易（下單、設定止損止盈）
步驟 10: 交易後記錄（更新統計、觸發報告）
```

### Know-How 要點

- **步驟失敗熔斷**：任一步驟失敗，後續步驟不執行，避免半完成狀態
- **步驟耗時監控**：每步驟記錄執行時間，識別性能瓶頸
- **可重入設計**：中途失敗可從失敗步驟重新開始，無需從頭執行
- **審計日誌**：每步驟輸出結構化日誌，完整保存決策鏈

## 七、多源外部數據整合與綜合情緒計算

**獨特性** ⭐⭐ | **可移植性** ⭐⭐⭐ | **商業價值** ⭐⭐

### 核心概念

整合 4 個免費外部數據源，計算**綜合市場情緒分數**，提供宏觀環境感知能力。

### 數據源與情緒權重

```
綜合情緒分數 =
  恐慌貪婪指數 (Alternative.me)   × 30%
  + 技術指標情緒                   × 30%
  + 市場動量                       × 25%
  + 新聞情緒 (NewsAPI)             × 15%
```

### 整合架構

```
Alternative.me → 恐慌貪婪指數 (0-100)
CoinGecko      → 全球市值、BTC/ETH 占比、穩定幣供應
DefiLlama      → DeFi TVL 趨勢
NewsAPI        → 新聞標題情感分析
       ↓
  WebDataFetcher（統一抓取 + 15 分鐘緩存 + 重試機制）
       ↓
  MarketAnalyzer（情緒融合計算）
       ↓
  TradingPlanController（納入交易決策）
```

### Know-How 要點

- **分層緩存機制**：15 分鐘緩存避免 API 頻率限制，同時保持數據新鮮度
- **異步並行抓取**：4 個數據源同時請求，總耗時等於最慢者（非總和）
- **失敗降級**：任一源失敗時使用緩存值，不影響整體決策
- **穩定幣占比作為風險指標**：穩定幣占比上升 → 市場避險情緒增加 → 調低多頭倉位

## 八、時光機回測引擎（防未來數據洩漏）

**獨特性** ⭐⭐⭐ | **可移植性** ⭐⭐⭐ | **商業價值** ⭐⭐⭐

### 核心概念

許多回測系統存在「未來數據洩漏」問題，導致回測結果過於樂觀。BioNeuronAI 的回測引擎採用**嚴格的時間隔離機制**，確保每個決策點只能看到當時可得的數據。

### 防洩漏機制

```
標準回測（危險）:
  t=100 的決策 → 可看到 t=1 到 t=200 的數據  ❌ 未來洩漏

BioNeuronAI 時光機（安全）:
  t=100 的決策 → 只能看到 t=1 到 t=100 的數據  ✅
  指標計算窗口在 t 點截斷
  外部數據只使用 t 時刻之前已發布的版本
```

### Walk-Forward 驗證框架

```
訓練期（9 個月）──────────── 驗證期（3 個月）
[Month  1 ~ Month  9]       [Month 10 ~ Month 12]
[Month  4 ~ Month 12]       [Month 13 ~ Month 15]
[Month  7 ~ Month 15]       [Month 16 ~ Month 18]
              ↓
     滾動式前推，評估策略泛化能力
```

### Know-How 要點

- **指標預計算快取**：在回測開始前一次性計算所有指標序列，避免每步重複計算，提升 10× 速度
- **1000 bars/s 回測速度**：向量化計算替代逐 bar 循環
- **詳細的 PnL 歸因**：拆解收益來源（策略 alpha / 市場 beta / 手續費損耗）
- **蒙地卡羅模擬**：對回測結果進行 1000 次隨機擾動，評估策略的統計顯著性

## 十、自研雙語迷你 LLM + RAG 知識庫

### 核心概念

BioNeuronAI 內建**自研 100M 參數雙語（中英文）LLM**，搭配 **RAG（檢索增強生成）** 架構，提供基於知識庫的市場問答與分析能力。

### 架構組成

```
src/nlp/         — 自研 LLM 工具包（100M 參數 Tiny LLM）
src/rag/         — RAG 檢索增強生成引擎
model/
  tiny_llm_en_zh/          — 雙語基礎版
  tiny_llm_en_zh_trained/  — 加密貨幣領域微調版 ⭐
```

### RAG 流程

```
用戶查詢（中文/英文）
       ↓
sentence-transformer 向量化
       ↓
FAISS 向量資料庫檢索（Top-K 相關文件）
       ↓
上下文 + 問題 → Tiny LLM
       ↓
生成回答（包含來源引用）
```

### Know-How 要點

- **領域微調**：在加密貨幣交易文本上微調，理解幣圈專業術語（HODL、DeFi、L2 等）
- **雙語無縫切換**：同一模型處理中英文查詢，適合台灣/中國用戶
- **本地推論**：模型本地部署，數據不外傳，符合金融合規需求
- **FAISS 高速向量搜尋**：百萬級文件毫秒級檢索
