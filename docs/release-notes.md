# 版本說明

以下為 BioNeuronAI 主要版本的更新記錄。

## v0.4.0 — 2024-06-15
- ✅ 導入 **新穎性閘門 (Novelty Gate)**，支援即時評估輸入樣本的新穎程度並調整後續路徑。
- ✅ 新增 **安全模組 (Safety Module)**，提供策略審核、執行節流與工具操作驗證。
- ✅ 優化短期記憶池管理與 Hebbian 權重更新，延長模型穩定期 1.8 倍。

## v0.3.2 — 2024-04-28
- 改善 Hebbian 學習收斂速度並降低震盪。
- 釋出互動式 CLI，便於測試與 demo。

## v0.3.0 — 2024-03-10
- 引入多層 BioNet 預設架構，支援複合式任務建模。
- 提供標準化的測試集與評估流程。

## v0.2.1 — 2024-01-18
- 優化突觸權重初始化策略，減少訓練前期的偏態。
- 加入更多使用範例腳本。

## v0.2.0 — 2023-12-05
- 開放外部擴充 API，支援自定義突觸更新規則。
- 建立基本文件與開發指南。

過往版本請參考 Git 標籤與提交記錄。
