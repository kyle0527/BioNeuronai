## 快速概覽

### 策略一：RSI 背離 📊
**最適合**: 震盪市場、反轉交易
- 牛背離 → 買入
- 熊背離 → 賣出
- RSI < 30 → 超賣
- RSI > 70 → 超買

### 策略二：布林帶突破 📈
**最適合**: 趨勢市場、突破交易
- 突破上軌 + 量能 → 追多
- 跌破下軌 + 量能 → 反彈多
- 帶寬收縮 → 大行情前兆
- 遠離中軌 → 均值回歸

### 策略三：MACD 趨勢 📉
**最適合**: 趨勢市場、中長線
- 金叉 → 買入
- 死叉 → 賣出
- 零軸上方 → 多頭市場
- 柱狀圖擴張 → 動量增強

### AI 融合策略 🤖
**最適合**: 所有市場環境
- 智能融合三個策略
- 根據表現動態調整權重
- 自動學習最佳組合
- 多維度確認信號

## AI 如何自我進化

```
1. 收集數據 → 三個策略分別分析
2. 生成信號 → 記錄置信度和原因
3. 融合決策 → 加權投票選出最佳方向
4. 執行交易 → 監控盈虧
5. 學習優化 → 調整策略權重
6. 持續進化 → 適應市場變化
```

### 評分標準
- 40% 平均收益率
- 30% 勝率
- 30% 夏普比率（風險調整後收益）

### 權重更新
```
新權重 = 舊權重 × 70% + 策略評分 × 30%
```
平滑調整,避免劇烈變化

## 關鍵參數調整

### 如果信號太多（過度交易）
```python
RSI_PERIOD = 21  # 增加週期（默認 14）
BOLLINGER_STD_DEV = 2.5  # 增加標準差（默認 2.0）
MACD_FAST_PERIOD = 16  # 減慢反應（默認 12）
```

### 如果信號太少（錯過機會）
```python
RSI_PERIOD = 10  # 減少週期
BOLLINGER_STD_DEV = 1.8  # 減少標準差
MACD_FAST_PERIOD = 10  # 加快反應
```

### 如果假信號多（準確度低）
```python
ACTIVE_STRATEGY = "AI_Fusion"  # 使用融合策略
AI_ENABLE_DYNAMIC_WEIGHTS = True  # 啟用動態權重
AI_MIN_TRADES_FOR_ADJUSTMENT = 10  # 快速學習調整
```

## 風險管理

### 每筆交易
- 止損: 2-3%
- 止盈: 4-6% (2:1 風險回報)
- 最大倉位: 賬戶的 5-10%

### 整體控制
- 最大同時持倉: 3 個
- 單日最大交易次數: 10 次
- 最大回撤: 10% (停止交易)

## 成功關鍵

### ✅ 要做的
1. **充分測試** - 至少測試網運行 2 週
2. **記錄分析** - 記錄每筆交易的理由和結果
3. **嚴格紀律** - 遵守止損,不衝動交易
4. **持續學習** - 分析失敗案例,優化策略
5. **風險第一** - 永遠不要冒超過承受範圍的風險

### ❌ 不要做的
1. **盲目相信** - 任何策略都會失效
2. **過度交易** - 不是每個信號都要做
3. **重倉賭博** - 單次交易不超過賬戶 10%
4. **忽視止損** - 止損是保護資金的最後防線
5. **追求完美** - 50-60% 勝率已經很好
