# 強化學習案例研究：探索增益最佳化

## 背景
一家自主機器人新創在高維度環境中訓練策略時，遭遇探索效率低落與狀態回放重複的瓶頸。既有的 epsilon-greedy 策略無法迅速導向未觀測的狀態空間，導致訓練成本飆升。

## 解決方案
- 將 BioNeuronAI 新穎性門控整合至策略網路的中介層，用以評估狀態-動作對的新穎性。
- 使用 `BioNeuron.novelty_score()` 輸出作為探索增益 (exploration bonus)，在 PPO 演算法的獎勵函式內動態調整。
- 透過記憶長度設定，讓代理得以逐回合調整對近期狀態的敏感度。

## 成效
- 訓練 48 小時後，策略成功探索到 35% 以上的未訪狀態，比基準方法高出 18%。
- 整體樣本效率提升 22%，收斂迭代數減少至原方案的 2/3。
- 透過 BioNeuronAI 的可調權重與 CLI 模擬器，團隊得以快速迭代多個環境設定。

## 未來展望
- 依據 0.2.0 里程碑，計畫引入原生探索增益 API 與診斷儀表板，降低客製整合成本。
