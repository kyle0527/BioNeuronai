## 🎓 使用場景

### 場景 1: 資源受限設備（邊緣設備）
```python
# 8-bit 量化 + LoRA
model = quantize_model(model, QuantizationConfig(bits=8))
model = apply_lora_to_model(model, LoRAConfig(r=8))
# 結果: 內存使用 <10% 原始大小
```

### 場景 2: 高吞吐量服務（API 服務器）
```python
# 批量推理 + KV Cache
engine = BatchInferenceEngine(model, BatchInferenceConfig(batch_size=32))
# 結果: 吞吐量提升 3-4x
```

### 場景 3: 實時對話（聊天機器人）
```python
# 流式生成 + KV Cache
generator = StreamingGenerator(model)
for token in generator.generate_stream(input_ids, use_cache=True):
    display(token)
# 結果: 零延遲感知
```

### 場景 4: 快速微調（垂直領域）
```python
# LoRA + 混合精度
model = apply_lora_to_model(model, LoRAConfig(r=8))
trainer = Trainer(model, TrainingConfig(use_amp=True))
# 結果: 訓練時間和成本 <5%
```

### 場景 5: 跨平台部署（多端部署）

```python
# 導出多種格式
export_model_package(model, export_formats=["onnx", "torchscript"])

```
