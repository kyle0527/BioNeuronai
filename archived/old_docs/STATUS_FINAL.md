## ✅ 錯誤修復記錄

### 已修復的問題

#### 1. Top-K 超出範圍錯誤 ✅
**問題**: `RuntimeError: selected index k out of range`
**原因**: vocab_size < k 時，topk 失敗
**修復**: 添加 `actual_k = min(k, probs.size(-1))`
**文件**: `uncertainty_quantification.py`

#### 2. TinyLLMConfig 參數名錯誤 ✅
**問題**: `TypeError: unexpected keyword argument 'hidden_dim'`
**原因**: 使用了錯誤的參數名
**修復**: 改為正確的參數名 `embed_dim`
**文件**: `honesty_demo.py` (3處)

#### 3. AssertionError: embed_dim % num_heads ✅
**問題**: 沒有指定 num_heads，使用默認值不匹配
**原因**: 缺少 num_heads 參數
**修復**: 添加 `num_heads=4` 參數
**文件**: `honesty_demo.py` (demo_6)

#### 4. Unicode 編碼錯誤 ✅
**問題**: `UnicodeEncodeError: 'cp950' codec can't encode character`
**原因**: 特殊 Unicode 字符在 Windows 終端
**修復**: 替換所有特殊字符為 ASCII 兼容字符
**文件**: `honesty_demo.py` (多處)
