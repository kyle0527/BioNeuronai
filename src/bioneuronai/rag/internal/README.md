## 📁 核心文件

### `knowledge_base.py`
知識庫管理器，負責知識庫的創建、更新和查詢。

**主要類**:
- `KnowledgeBase` - 知識庫管理器

**核心功能**:
- 知識添加
- 知識更新
- 知識刪除
- 知識查詢

## 🛠️ 主要功能

### 1. 知識存儲
將文檔和元數據存儲到知識庫。

### 2. 索引管理
維護文檔索引以支持快速檢索。

### 3. 版本控制
追蹤知識庫的變更歷史。
