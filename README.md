# BioNeuronAI

BioNeuronAI is a bio-inspired novelty gating toolkit with Hebbian learning, temporal memory, and safety modules for AI assistants.

- [English](#english)
- [中文](#中文)

## English

### ✨ Features
- **Hebbian learning** with configurable stability bounds and short-term memory windows.
- **Novelty detection** that scores surprising inputs for tool and pipeline gating.
- **Composable networks** that stack neurons into layers and higher level controllers.
- **Security integrations** (SQLi/IDOR/auth) that protect production assistants.
- **Smart Learning Assistant** that reviews repositories and suggests next steps.

### 🚀 Quickstart
```bash
# Clone the repository
git clone https://github.com/kyle0527/BioNeuronai.git
cd BioNeuronai

# Install (development mode)
pip install -e .

# Install development extras (formatters, linters, docs)
pip install -e .[dev,docs]
```

#### Basic Usage
```python
from bioneuronai.core import BioNeuron, BioNet

neuron = BioNeuron(num_inputs=2, threshold=0.6, learning_rate=0.05)
output = neuron.forward([0.8, 0.6])
print(f"Output: {output}")

neuron.hebbian_learn([0.8, 0.6], target_output=1.0)
print(f"Novelty score: {neuron.novelty_score()}")

net = BioNet()
l2_out, l1_out = net.forward([0.5, 0.8])
net.learn([0.5, 0.8])
```

#### Command Line Interface
```bash
bioneuron-cli
```
