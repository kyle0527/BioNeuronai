# BioNeuronAI

This repository contains the core implementation of BioNeuronAI, a bio-inspired novelty gate for AI assistants. It provides a Hebbian neuron model with short-term input memory and a simple novelty score, plus scaffolding to integrate reinforcement learning loops, retrieval-augmented generation (RAG) pipelines, and tool gating.

This branch initializes the project with a basic README; subsequent commits will add source code, documentation, and tests.
